# StarterKit

YU ZIJIE's Starter Kit For Creating Websites.

## Inspired By

* [HTML5 Boilerplate][1]
* [normalize.css][2]
* [Bourbon][3]
* [DevTips][7]
* and some of my own collections

## System Requirement

* Bash
* [NodeJS][4]
* [Ruby][5]
  * [SASS][6]

## Dependencies

* Modernizr
* jQuery
* Bourbon / Neat
* [jQuery Mouse Wheel Plugin][8] (required by prevent-scroll.js)

## How to use

1. run: `npm install`
2. Take a look at `/test` folder
